// METHODOLOGY
How AutoVal AI Works
01
📥
Data Collection
Pakistan car data from Kaggle (PakWheels/OLX listings) with features: brand, model, year, mileage, fuel, transmission, condition, city.
02
⚙️
Preprocessing
Raw data is cleaned, missing values handled, categories label-encoded, and features scaled for ML compatibility.
03
🧠
Model Training
Three algorithms trained: Linear Regression (baseline), Random Forest, and Gradient Boosting. Best R² model selected.
04
💾
Model Saved
Best model saved as .pkl using joblib. Encoders and scaler also saved so predictions are consistent at inference time.
05
🔌
Flask API
A Python Flask backend loads the model and exposes a /predict endpoint. The frontend sends car data as JSON and receives a price.
06
🎯
Live Prediction
Your inputs are encoded, scaled, and passed through the model. The result includes price, confidence range, and factor analysis.
